Pyqi文档
===============
介绍
-------
Pyqi(Python3 quick installer)，能够帮你更快的安装Python3。本软件使用Python编写，使用Pyinstaller编译，使未安装Python的设备也能运行，传统安装方法需要大约10分钟，而本程序安装只需要约三分钟。

本软件通过预编译以及镜像下载等方法多方面加速安装进程。以提高安装速度。
